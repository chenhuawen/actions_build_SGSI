使用方法:
1. fork此仓库
2. 编辑sgsi.json:
     rom_url: 为你的下载连接 
     rom_name: 为你下载的包名 
     pack_sgsi: 为你需要打包的包名
     build_type: 为你需要build的类型(AB|A) 
     rom_type: 为你要跑的rom类型  
     light_fix: 是否进行亮度修复
     bug_fix: 是否进行bug修复（仅支持MIUI,Flyme,H2OS,Color,请输入小写）
     use_7zip: 使用 7z 压缩 SGSI
     use_zip: 使用 zip 压缩 SGSI
3. 修改完毕后依次点击Start commit -> Actions选项卡 -> Star -> build_SGSI-10
